# Feynman's Perpetual Motion Argument

Feynman proves that perpetual motion is impossible without examining the internals of any machine. Instead, he shows what would follow if it were possible, and demonstrates that the consequences are absurd. The method is proof by contradiction applied to physics, and its elegance is in what it does not need to know.

The setup involves two machines. Machine A is reversible — it can lift a weight to height X, and the process can be run in reverse to restore the original state. Machine B is irreversible — it lifts the same weight to height Y, but cannot be reversed.

Suppose Y is greater than X. Then Machine B lifts the weight to Y, and it is simply dropped from Y to X — using nothing but gravity. That drop is free energy: the weight's potential energy decreases, and the difference is captured. Then Machine A, which is reversible, restores everything to the starting configuration. The system has returned to its original state, and energy has been extracted for nothing. Repeating this process indefinitely yields perpetual motion.

A common point of confusion is thinking Machine B must be involved in lowering the weight from Y to X, which would require energy since B is irreversible. But Machine B plays no role in the descent. The weight is simply dropped — gravity does the work. That gravitational descent is where the free energy comes from. Machine A then resets the system.

Since perpetual motion is impossible, the assumption must be wrong. Y cannot be greater than X. If both machines are reversible, the argument can be run in either direction, which means Y must equal X exactly. No reversible machine can outperform any other reversible machine — they are all equally efficient. This is a foundational result in thermodynamics, derived from pure logic without any knowledge of the machines' internal mechanisms.

The beauty of the argument is its generality. It does not require knowing how either machine works. It requires only two premises: that one machine is reversible and that perpetual motion is impossible. From those alone, the constraint on relative performance follows necessarily. It is physics derived from logic rather than experiment — though experiment confirms it.
